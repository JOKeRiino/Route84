Route84

--> If the game is lagging, open the pause menu and click "resume" <--

This game serves as a simple representation and showcase for the JavaScript library "kaboom.js."
It runs on JavaScript only and is rendered by the HTML canvas element. Almost all sprites in this
game were made by me. This is my first try at making a (browser) game.

Currently, Route84 is in alpha stage (Version: 0.7.1), therefore has some bugs and does not
feature mobile support.

If you find any bugs you can tell me on Github. (And also follow the project if interested!)
If you want to give general feedback, you can do that on Twitter.
Also, if you enjoyed the game, you can send me some doge coin here.

How To Play

  • Your goal is to refuel the cars. If a car stops at a gas pump, click on it to reveal the
    refueling minigame.
  • To beat the minigame, spam [SPACEBAR] repeatedly and keep the yellow fuel level within the
    green bar (Watch out for the BORDERS!).
  • Buy upgrades to level up and therefore earn more money from fueling up cars.
  • The Clerk is a donkey.
  • Have fun :)
