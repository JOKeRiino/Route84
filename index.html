Route84

This game serves as a simple representation and showcase for the JavaScript library "kaboom.js."
It runs on JavaScript only and is rendered by the HTML canvas element. Almost all sprites in this
game were made by me. This is my first try at making a (browser) game.

Currently, Route84 is in alpha stage (Version: 0.7.0), therefore has some bugs and does not
feature mobile support.

If you find any bugs you can tell me on Github. (And also follow the project if interested!)
If you want to give general feedback, you can do that on Twitter.
